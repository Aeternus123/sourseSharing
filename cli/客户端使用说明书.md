# 数据共享区块链系统 - 客户端使用说明书

## 📋 产品概述

数据共享区块链系统是一个基于区块链技术的分布式文件共享平台，通过智能合约实现文件上传、下载、分享及虚拟币激励机制。本客户端提供简洁高效的命令行界面，方便用户管理文件和虚拟币。

## 🚀 安装指南

### 编译安装（开发者）

1. 确保已安装 Go 开发环境
2. 进入 cli 目录
3. 双击运行 `build.bat` 文件
4. 编译成功后会生成 `sscli.exe` 可执行文件

### 直接使用

直接使用目录中已编译好的 `sscli.exe` 文件。

## ⚙️ 快速开始

### 1. 设置服务器连接

```bash
sscli config server http://您的服务器IP:8080
```

执行后系统会自动测试连接，显示连接状态。

### 2. 用户登录

```bash
sscli login 0x您的以太坊地址
```

系统将自动使用默认密码进行登录认证。

### 3. 查看用户信息

```bash
sscli info
```

显示您的账户余额、存储使用情况、收入和支出等信息。

### 4. 上传文件

```bash
sscli upload C:\路径\到\您的文件.pdf
```

默认情况下，上传的文件将自动设置为共享状态。若要上传私有文件：

```bash
sscli upload C:\路径\到\您的文件.pdf --private
```

### 5. 下载文件

```bash
sscli download 0x文件所有者地址 文件哈希 [保存路径]
```

- 保存路径为可选参数，若不指定则使用文件名保存到当前目录

### 6. 查看共享文件市场

```bash
sscli shared
```

浏览系统中所有可下载的共享文件。

## 📁 文件管理命令

### 上传文件

```bash
sscli upload <文件路径> [--private]
```

- `<文件路径>`: 要上传的文件的完整路径
- `--private`: 可选参数，设置文件为私有状态

上传成功后，系统会显示文件哈希、名称、大小和分享状态等信息。

### 下载文件

```bash
sscli download <所有者地址> <文件哈希> [保存路径]
```

- `<所有者地址>`: 文件所有者的以太坊地址
- `<文件哈希>`: 文件的唯一标识符
- `[保存路径]`: 可选，指定保存位置

下载成功后，系统会自动识别原始文件名并重命名。

### 删除文件

```bash
sscli delete <文件哈希>
```

执行此命令后，系统会要求确认操作。删除操作不可逆，请谨慎使用。

### 列出用户文件

```bash
sscli list [用户地址]
```

- `[用户地址]`: 可选，查看指定用户的文件列表，若不指定则显示当前登录用户的文件

显示文件的名称、大小、状态、下载次数和哈希值。

### 查看共享文件市场

```bash
sscli shared
```

显示所有可下载的共享文件，包括文件名、大小、价格、下载次数、所有者和哈希值。

## 👤 用户管理命令

### 配置服务器地址

```bash
sscli config server <服务器URL>
```

例如：
```bash
sscli config server http://localhost:8080
```

### 用户登录

```bash
sscli login <以太坊地址>
```

### 用户退出登录

```bash
sscli logout
```

### 查看用户信息

```bash
sscli info
```

显示当前登录用户的详细信息，包括：
- 以太坊地址
- 账户余额（虚拟币）
- 存储使用情况（已用/总容量）
- 总收入和总支出
- 账户状态（激活/未激活）

### 查看当前配置

```bash
sscli config
```

显示服务器地址、连接状态、当前用户地址和管理员地址。

### 管理员退出登录

```bash
sscli admin-logout
```

```bash

```


为新用户创建账户并初始化虚拟币余额。

## 💰 虚拟币机制说明

### 虚拟币获取方式

1. **注册奖励**：新用户注册时获得初始虚拟币（当前为4个）

2. **上传奖励**：
   - 基础上传奖励：2个虚拟币
   - 文件大小奖励：每MB额外获得1个虚拟币
   - 例如：上传一个5MB的文件，将获得2 + 5 = 7个虚拟币

3. **下载收益**：每次您的共享文件被其他用户下载，您将获得1个虚拟币

### 虚拟币消耗方式

1. **下载费用**：下载其他用户的共享文件时，需要支付1个虚拟币

## 📊 系统限制

- 单用户存储空间限制：1GB
- 每个文件的下载价格：1个虚拟币（系统默认）
- 文件最大大小：取决于服务器配置

## ⚠️ 注意事项

1. **私钥安全**：确保保护好您的以太坊私钥，不要分享给他人
2. **文件备份**：系统不保证文件永久保存，请做好本地备份
3. **网络连接**：确保客户端能正常连接到服务器
4. **防火墙设置**：如遇连接问题，请检查防火墙设置

## ❓ 常见问题

**Q: 如何获取我的以太坊地址？**
A: 您可以通过MetaMask等以太坊钱包软件获取您的地址，格式通常为以"0x"开头的42个字符的字符串。

**Q: 文件上传失败怎么办？**
A: 请检查以下几点：
- 服务器连接是否正常
- 文件路径是否正确
- 文件大小是否超过限制
- 您的存储空间是否已满

**Q: 忘记了用户地址怎么办？**
A: 执行 `sscli config` 命令，在配置信息中可以查看到当前登录的用户地址。

**Q: 如何查看已上传文件的哈希值？**
A: 使用 `sscli list` 命令查看您的文件列表，其中包含每个文件的哈希值。
